# 우왘쿹 코멤 아카테미 "친희" 말투 편환키

이 프로쳌트는 우왘쿹 코멤 아카테미의 "친희"님의 말투를 편환하는 키켸잎니타. 이 프로쳌트는 이러한 말투를 차통으로 편환해추는 윂 어플리케이션잎니타.

## 사용 팡펖

1. 사이트 첲솤핲니타.
2. 텤스트 잎렼란에 편환하코 싶은 한클 문창을 잎렼핲니타.
3. "편환" 퍼튼을 클맄핲니타.
4. 편환퇸 켤콰카 아래에 표시툎니타.

## 사용 출처

- [hangeul.js](https://github.com/e-/Hangul.js)
